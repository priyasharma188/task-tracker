# 📌 Task Tracker (MERN Stack)

A modern Task Tracker application built using the MERN Stack. This project allows users to create, view, update, and delete tasks through a clean and responsive user interface.

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* 🌙 Dark/Light Mode
* 📱 Responsive Design
* 🔗 REST API Integration
* 💾 MongoDB Database

## 🛠️ Tech Stack

### Frontend

* React.js
* Bootstrap 5
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/priyasharma188/task-tracker.git
```

### Install dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

### Environment Variables

Create a `.env` file inside the **backend** folder.

Example:

```
MONGO_URI=your_mongodb_connection_string
PORT=3000
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm start
```
